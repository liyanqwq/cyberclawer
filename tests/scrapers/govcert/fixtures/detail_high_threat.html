High Threat Security Alert (A26-06-01): Vulnerability in Linux Kernel

Published on: 01 June 2026

Description:

A local privilege escalation vulnerability (CIFSwitch) is found in the Linux kernel.

Reports indicated that proof-of-concept (PoC) exploit code is publicly available.

 

Affected Systems:

  • Linux kernel version with the CIFS client capability and the cifs-utils package version 6.14 or high

 

Impact:

Successful exploitation of the vulnerability could lead to elevation of privilege.

 

Recommendation:

System administrators should follow vendor recommendations.
